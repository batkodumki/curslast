# Модуль експорту візуальних порівнянь

## Опис

Модуль `export_visuals.py` автоматично генерує PNG зображення терезів для всіх рівнів впливу експертного оцінювання.

## Використання

### Пакетна генерація всіх зображень

```bash
python3 export_visuals.py
```

Команда згенерує всі 9 зображень у папці `exported_visuals/`.

### Використання в коді

```python
from export_visuals import export_all_visuals, BalanceScaleVisualizer

# Пакетна генерація всіх зображень
export_all_visuals(output_dir="exported_visuals")

# Створення окремого зображення
visualizer = BalanceScaleVisualizer(width=800, height=600)
img = visualizer.create_visual("Сильно", 5.0)
img.save("custom_output.png", "PNG")
```

## Згенеровані файли

Усі файли мають розмір 800×600 пікселів, білий фон та українські підписи:

1. `slabko-abo-neznachno.png` - Слабко або незначно (значення: 1.5)
2. `serednie.png` - Середнє (значення: 3.0)
3. `bil-she-nizh-serednie.png` - Більше ніж середнє (значення: 4.5)
4. `sylno.png` - Сильно (значення: 5.0)
5. `bil-she-nizh-sylno.png` - Більше ніж сильно (значення: 6.0)
6. `duzhe-sylno.png` - Дуже сильно (значення: 7.0)
7. `duzhe-duzhe-sylno.png` - Дуже-дуже сильно (значення: 8.0)
8. `absolutno.png` - Абсолютно (значення: 9.0)
9. `absolutna-perevaha.png` - Абсолютна перевага (значення: 9.5)

## Візуалізація

Кожне зображення показує:
- **Терези з чашами**: Візуалізація балансу між двома альтернативами
- **3D кубики**: Розміщені НА чашах терезів, розмір залежить від інтенсивності впливу
- **Нахил терезів**: Відображає перевагу одного варіанту над іншим
- **Підпис**: Українська назва рівня впливу внизу зображення

### Кольорова схема
- Лівий кубик (синій): #4A90E2 - представляє базову альтернативу
- Правий кубик (коралевий): #FF6B6B - представляє порівнювану альтернативу
- Терези (сірий): #7F8C8D - нейтральний елемент порівняння

## Технічні деталі

- **Бібліотека**: Pillow (PIL)
- **Формат**: PNG
- **Розмір**: 800×600 px
- **Фон**: Білий (#FFFFFF)
- **Шрифт**: DejaVu Sans Bold, 28pt (якщо доступний)
